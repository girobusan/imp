IMP! Preconfigured for CDN

From here you can get page, preconfigured to use CDN for editor script and view CSS.

  1. Switch to edit mode.
  2. Scroll down, click the "Duplicate page" button
  3. Enter any filename you want (ending with .html), save.

This setup is ideal as starting point. You will get just one file, fully working and ready. Both editor and default theme will be served from CDN. If you will need to change default theme, you will also need to adjust View CSS location setting accordingly.
