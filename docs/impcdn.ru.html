  • Что такое IMP?

Здесь вы можете создать страничку, которая использует CDN для редактора и основного CSS.

  1. Переключитесь в режим редактирования.
  2. Промотайте вниз, нажмите на кнопку "Duplicate page"
  3. Введите желаемое имя файла с расширением html, сохраните где нужно.

Эта настройка — хороший способ начать работать с IMP! У вас будет всего один файл, полностью готовый к работе. И редактор, и тема оформления будут загружаться с CDN по необходимости. Но если вы захотите поменять тему, вам нужно будет соответственно поменять настройку View CSS location.
