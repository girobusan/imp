IMP! (It's My Page!)

Like a markdown-based CMS for single file, fully client-side. When you need just drop one page somewhere on the web, and do not want (can not) use full-blown CMS or SSG, but still want to use GUI and markdown for content editing.

This page is made with IMP!, you can see how it looks in edit mode.
